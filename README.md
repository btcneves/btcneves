## 🧩 Featured Projects

### [NodeScope](https://github.com/btcneves/NodeScope)
Bitcoin Core intelligence dashboard built with **Python, FastAPI, React, TypeScript, Docker, RPC, ZMQ, Prometheus, pytest, and GitHub Actions**.

Real-time observability layer for Bitcoin Core nodes, combining RPC snapshots, ZMQ streams, mempool monitoring, structured logs, transaction classification, metrics, and a reproducible technical validation workflow.

### [phantos-coldwallet](https://github.com/btcneves/phantos-coldwallet)
Bootable Bitcoin cold wallet focused on **offline signing, PSBT workflows, BIP39/BIP32 derivation, QR/UR transfer, watch-only exports, and air-gapped operation**.

Designed for secure cold storage, offline signing stations, and key recovery workflows.

### [CoreCraft](https://github.com/btcneves/CoreCraft)
Suite of Bitcoin Core applications covering **RPC mempool snapshots, real-time ZMQ monitoring, and multi-wallet PSBT workflows**.

Built with FastAPI, React, Docker Compose, pytest, mypy, GitHub Actions, and reproducible regtest environments.

### [AprovAI](https://github.com/btcneves/AprovAI)
Adaptive study and exam platform built with **React, TypeScript, Vite, Node.js, Vitest, ESLint, and pdfjs-dist**.

Includes question bank management, adaptive recommendations, analytics, anti-repetition filters, and PDF-based data ingestion.
